  <action>
Create scripts/lib/rolling-windows.ts. This module provides two exported functions that accept a db `sql` handle plus IDs and compute rolling window stats for a single team or player within a season. compute-stats.ts calls these for every team and player after advanced stats are computed.

**computeTeamRollingWindows(sql, teamId: string, seasonId: number): Promise&lt;void&gt;**
- Fetch all advanced_team_game_stats rows for this team + season, joined to games for game_date and is_win derivation, ordered by game_date ASC
- Filter: only games with status = 'Final' AND game_team_box_scores row exists (use the JOIN to advanced_team_game_stats as implicit filter — if we computed advanced stats for a game, a box score row existed)
- Derive is_win: `CASE WHEN g.home_team_id = teamId THEN g.home_score > g.away_score ELSE g.away_score > g.home_score END`
- Slide windows [5, 10]: for each game index i, if i+1 >= window, take slice [i+1-window, i+1]
- For the slice, compute averages of off_rating, def_rating, net_rating, pace, efg_pct, ts_pct, tov_pct, reb_pct; count wins/losses
- as_of_game_date = the last game's game_date in the slice (rows[i].game_date)
- Upsert into rolling_team_stats with ON CONFLICT (team_id, season_id, window_games, as_of_game_date) DO UPDATE SET ...

**computePlayerRollingWindows(sql, playerId: string, teamId: string, seasonId: number): Promise&lt;void&gt;**
- Fetch all advanced_player_game_stats rows for this player + join to games for game_date, ordered ASC
- Filter: only games where the player row exists in advanced_player_game_stats (already filtered by the query)
- Also join game_player_box_scores to get raw pts/reb/ast/minutes values for the window averages (rolling_player_stats stores raw counts, not advanced stats, for pts/reb/ast/minutes)
- Slide windows [5, 10]: same approach as teams
- For the slice, compute: avg(points), avg(rebounds), avg(assists), avg(ts_pct), avg(efg_pct), avg(minutesDecimal)
- minutesDecimal: use parseMinutes on game_player_box_scores.minutes
- Upsert into rolling_player_stats with ON CONFLICT (player_id, season_id, window_games, as_of_game_date) DO UPDATE SET ...

**1. Create scripts/compute-stats.ts**

Mirror the structure and style of scripts/backfill.ts. The orchestrator:
1. Queries all games with box score data (filter by EXISTS in game_team_box_scores)
2. For each such game: fetches both team box score rows, computes all team advanced stats, upserts to advanced_team_game_stats; then fetches all player box score rows for that game and computes + upserts player advanced stats
3. After all game-level stats are written, queries all distinct (team_id, season_id) pairs that appear in advanced_team_game_stats, calls computeTeamRollingWindows for each
4. Queries all distinct (player_id, team_id, season_id) pairs from advanced_player_game_stats joined to games, calls computePlayerRollingWindows for each
5. Prints final summary counts from all four derived tables

**Compute sequence (must respect dependency order):**
- Step 1: Team advanced stats (requires: game_team_box_scores)
- Step 2: Player advanced stats (requires: game_team_box_scores for team context + game_player_box_scores)
- Step 3: Team rolling windows (requires: advanced_team_game_stats from Step 1)
- Step 4: Player rolling windows (requires: advanced_player_game_stats from Step 2)

**Team advanced stats computation per game:**
```typescript
// Fetch both team box scores for the game
const boxes = await sql<TeamBoxRow[]>`
  SELECT
    b.team_id::text,
    b.is_home,
    b.points, b.fg_made, b.fg_attempted,
    b.fg3_made, b.fg3_attempted,
    b.ft_made, b.ft_attempted,
    b.offensive_reb, b.defensive_reb,
    b.turnovers, b.assists, b.rebounds
  FROM game_team_box_scores b
  WHERE b.game_id = ${gameId}::bigint
`;
if (boxes.length !== 2) {
  console.warn(`  Skipping game ${gameId}: expected 2 team boxes, got ${boxes.length}`);
  continue;
}

const [teamA, teamB] = boxes;
// Compute for each team: possessions, pace, ratings, efg, ts, tov, reb
// teamA's def_rating = teamB's off_rating (opponent scored points against teamA's possessions)
const possA = computeTeamPossessions({ fg_attempted: teamA.fg_attempted, ft_attempted: teamA.ft_attempted, offensive_reb: teamA.offensive_reb, turnovers: teamA.turnovers });
const possB = computeTeamPossessions({ ... teamB ... });
const pace = computePace(possA, possB); // same for both teams in same game

for (const [team, opp, poss, oppPoss] of [[teamA, teamB, possA, possB], [teamB, teamA, possB, possA]]) {
  const offRating = computeOffRating(team.points, poss);
  const defRating = computeOffRating(opp.points, oppPoss); // opponent scored against this team's defense
  const netRating = offRating - defRating;
  const efgPct = computeEfgPct(team.fg_made, team.fg3_made, team.fg_attempted);
  const tsPct = computeTsPct(team.points, team.fg_attempted, team.ft_attempted);
  const tovPct = computeTovPct(team.turnovers, team.fg_attempted, team.ft_attempted);
  const rebPct = computeRebPct(team.offensive_reb, team.defensive_reb, opp.offensive_reb, opp.defensive_reb);
  // upsert into advanced_team_game_stats
}
```

// Fetch all player rows for this game along with their team box score for context
const playerRows = await sql<PlayerBoxRow[]>`
  SELECT
    p.player_id::text, p.team_id::text,
    p.minutes, p.points, p.rebounds, p.assists, p.turnovers,
    p.fg_made, p.fg_attempted, p.fg3_made, p.fg3_attempted,
    p.ft_made, p.ft_attempted,
    p.offensive_reb, p.defensive_reb
  FROM game_player_box_scores p
  WHERE p.game_id = ${gameId}::bigint
`;

// Build a map of team_id -> team box score for context (already fetched above as `boxes`)
const teamBoxMap = new Map(boxes.map(b => [b.team_id, b]));
const oppBoxMap = new Map(boxes.map(b => [b.team_id, boxes.find(x => x.team_id !== b.team_id)!]));

for (const p of playerRows) {
  const teamBox = teamBoxMap.get(p.team_id);
  const oppBox = oppBoxMap.get(p.team_id);
  if (!teamBox) continue;
  const minutesDec = parseMinutes(p.minutes);
  const teamMinutes = 240; // regulation default
  const usageRate = computeUsageRate(
    { fg_attempted: p.fg_attempted, ft_attempted: p.ft_attempted, turnovers: p.turnovers, minutesDecimal: minutesDec },
    { fg_attempted: teamBox.fg_attempted, ft_attempted: teamBox.ft_attempted, turnovers: teamBox.turnovers, minutesDecimal: teamMinutes }
  );
  const efgPct = computeEfgPct(p.fg_made, p.fg3_made, p.fg_attempted);
  const tsPct = computeTsPct(p.points, p.fg_attempted, p.ft_attempted);
  const tovRate = computeTovPct(p.turnovers, p.fg_attempted, p.ft_attempted);
  const astRate = computeAstRate(
    { assists: p.assists, fg_made: p.fg_made, minutesDecimal: minutesDec },
    { fg_made: teamBox.fg_made, minutesDecimal: teamMinutes }
  );
  const rebRate = computeRebRate(
    { rebounds: p.rebounds, minutesDecimal: minutesDec },
    { rebounds: teamBox.rebounds, minutesDecimal: teamMinutes },
    oppBox ? oppBox.rebounds : 0
  );
  // upsert into advanced_player_game_stats
}
