<!-- Key contracts. -->

insights table (actual table name — NOT generated_insights):
- insight_id (uuid)
- scope (text): 'live' | 'between_games' | 'historical'
- category (text): 'streak' | 'milestone' | 'rare_event' | 'opponent_context' | 'league_comparison' | 'run' | 'clutch'
- headline (text)
- detail (text)
- importance (int 0–100)
- is_active (boolean) — only true rows are eligible
- game_id (bigint nullable) — filter by this when game_id param provided
- player_id (bigint nullable) — filter by this when player_id param provided
- proof_sql (text), proof_params (jsonb), proof_result (jsonb)
- created_at (timestamptz)

API_SPEC.md /api/insights response shape:
```json
{
  "meta": { ... },
  "insights": [
    {
      "insight_id": "uuid",
      "scope": "live",
      "category": "streak",
      "headline": "...",
      "detail": "...",
      "importance": 78,
      "proof": { "summary": "home_last9_record", "result": { "wins": 7, "losses": 2 } }
    }
  ]
}
```

scope param: required — one of "live" | "between_games" | "historical"
proof.summary = category field from DB
proof.result = proof_result JSONB field from DB (already parsed by postgres driver)

Query params from API_SPEC.md:
- scope (required)
- team (optional, default LAC — for MVP ignored, all insights are LAC-centric)
- game_id (optional, recommended for scope=live)
- player_id (optional)
- limit (optional, default 10, max 50)

  <action>
    Create src/app/api/insights/route.ts:

    1. Imports: NextResponse, sql, buildMeta, buildError

    2. Parse query params:
    ```typescript
    const url = new URL(request.url);
    const scope = url.searchParams.get('scope');
    const gameId = url.searchParams.get('game_id');
    const playerId = url.searchParams.get('player_id');
    const limitParam = Math.min(parseInt(url.searchParams.get('limit') ?? '10', 10) || 10, 50);
    ```

    3. Validate scope:
    ```typescript
    const validScopes = ['live', 'between_games', 'historical'];
    if (!scope) return NextResponse.json(buildError('BAD_REQUEST', 'scope param is required'), { status: 400 });
    if (!validScopes.includes(scope)) return NextResponse.json(buildError('BAD_REQUEST', `scope must be one of: ${validScopes.join(', ')}`), { status: 400 });
    ```

    4. SQL query with optional game_id and player_id conditions:
    ```typescript
    const rows = await sql`
      SELECT
        insight_id::text AS insight_id,
        scope,
        category,
        headline,
        detail,
        importance,
        proof_result
      FROM insights
      WHERE is_active = true
        AND scope = ${scope}
        ${gameId ? sql`AND game_id = ${gameId}::bigint` : sql``}
        ${playerId ? sql`AND player_id = ${playerId}::bigint` : sql``}
      ORDER BY importance DESC, created_at DESC
      LIMIT ${limitParam}
    `;
    ```

    5. Map to response shape:
    ```typescript
    const insights = rows.map(row => ({
      insight_id: row.insight_id,
      scope: row.scope,
      category: row.category,
      headline: row.headline,
      detail: row.detail,
      importance: row.importance,
      proof: {
        summary: row.category,
        result: row.proof_result,
      },
    }));
    ```

    6. Return:
    ```typescript
    return NextResponse.json(
      { meta: buildMeta('db', 30), insights },
      { headers: { 'Cache-Control': 'public, max-age=30' } }
    );
    ```
