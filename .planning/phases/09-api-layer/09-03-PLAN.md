    ```typescript
    let LAC_TEAM_ID_INTERNAL: bigint | null = null;
    async function getLacTeamId() {
      if (LAC_TEAM_ID_INTERNAL) return LAC_TEAM_ID_INTERNAL;
      const [row] = await sql`SELECT team_id FROM teams WHERE nba_team_id = ${LAC_NBA_TEAM_ID}`;
    ```

    3. Run parallel DB queries using Promise.all for performance (hit < 300ms SLA):
       - Query A: rolling_team_stats for LAC (most recent window_games=10 row)
       - Query B: last 10 LAC final games from games table (for last_10 W/L record)
       - Query C: current season record (all LAC final games this season)
       - Query D: upcoming schedule (future LAC games)
       - Query E: player trends from game_player_box_scores
       - Query F: between_games insights

    4. Build team_snapshot:
    ```typescript
    const currentSeason = new Date().getMonth() < 6
      ? new Date().getFullYear() - 1
      : new Date().getFullYear();
    ```
    Season_id = currentSeason (matches Phase 7 decision for season derivation).

    For record: count final games WHERE season_id = currentSeason.
    For last_10: count W/L from 10 most recent final games (any season).
    For ratings: take first rolling_team_stats row or null.

    5. upcoming_schedule: for each game, determine opponent_abbr (if home_team_id=LAC then away team is opp), home_away ('home'/'away'). Call getLatestOdds per game. Limit to first 10 upcoming.

    6. player_trends: aggregate from game_player_box_scores.
    ```sql
    SELECT
      p.player_id::text,
      p.display_name AS name,
      COUNT(*) AS game_count,
      AVG(gpbs.minutes_decimal)::float8 AS minutes_avg,
      AVG(gpbs.points)::float8 AS pts_avg,
      AVG(gpbs.rebounds_total)::float8 AS reb_avg,
      AVG(gpbs.assists)::float8 AS ast_avg,
      AVG(gpbs.ts_pct::float8) AS ts_pct
    FROM game_player_box_scores gpbs
    JOIN players p ON p.player_id = gpbs.player_id
    WHERE gpbs.team_id = ${lacTeamId}
      AND gpbs.game_id IN (
        SELECT game_id FROM games
        WHERE (home_team_id = ${lacTeamId} OR away_team_id = ${lacTeamId})
          AND status = 'final'
        ORDER BY game_date DESC LIMIT 10
      )
    GROUP BY p.player_id, p.display_name
    ORDER BY minutes_avg DESC
    LIMIT 8
    ```
    NOTE: Check DB_SCHEMA for exact column name for minutes in game_player_box_scores — may be 'minutes' (text "PT25M") needing extraction, or 'minutes_played' (numeric). Parse accordingly.

    7. Map insights: proof: { summary: row.category, result: row.proof_result }
