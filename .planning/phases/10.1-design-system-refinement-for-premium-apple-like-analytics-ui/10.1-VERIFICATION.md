### Human Verification Required

#### 1. Visual Premium Aesthetic

**Test:** Open the application in a browser and review all components against the Apple Sports / Linear / Raycast reference aesthetic
**Expected:** Restrained, information-dense, calm dark UI — no decorative elements, invisible borders that are visible in context but not assertive, high-contrast stat values
**Why human:** Visual quality and perceived "premium-ness" cannot be verified by code inspection

#### 2. TopNav Glass Blur

**Test:** Navigate to any page and scroll — observe the nav over page content
**Expected:** bg-background/80 backdrop-blur-md creates a frosted glass effect; content beneath the nav is visible but blurred
**Why human:** backdrop-filter CSS rendering is compositor-dependent and varies by browser/GPU

#### 3. StatCard Typographic Hierarchy

**Test:** View a stat card grid (e.g., Home page) at normal reading distance
**Expected:** The large bold number (1.75rem) immediately reads as the primary piece of information; the 11px ALL CAPS label above and 12px caption below are clearly secondary
**Why human:** Typographic hierarchy is a perceptual quality — the math is correct but human judgement confirms the reading order feels right
