## Decisions Made

- **StatCard plain div over Card primitive:** shadcn Card applies its own CSS variable remapping (--card, --card-foreground) that can override the new tokens unexpectedly. A plain div with explicit Tailwind utilities gives deterministic control.
- **rgba() for border/muted:** Opacity-based values (rgba(255,255,255,0.06)) look lighter and more "floating" on dark backgrounds than opaque hex equivalents, which is the premium aesthetic target.
- **text-positive/text-negative:** Switched from `text-[var(--positive)]` inline var() to Tailwind utility class approach, consistent with how the rest of the system uses @theme inline registered colors.
