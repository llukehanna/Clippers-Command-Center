## Sampling Rate

- **After every task commit:** Browser visual check at 1024px and 1440px widths — verify target component matches CONTEXT.md spec
- **After every plan wave:** Full page review with all components visible; run `npx vitest run` to confirm no regressions
- **Before `/gsd:verify-work`:** Visual sign-off comparing each component against reference apps (Apple Sports, Linear, Raycast)
- **Max feedback latency:** One browser reload per task
