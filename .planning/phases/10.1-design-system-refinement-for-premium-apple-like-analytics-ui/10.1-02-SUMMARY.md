## Decisions Made

- ChartSeries.color made optional: callers that already pass `var(--chart-1)` continue to work; new callers get correct defaults without needing to know the token names
- strokeOpacity used instead of opacity: `opacity` fades the entire SVG grid group (including axis labels that share the group), `strokeOpacity` targets only the grid line stroke — precision required for the near-invisible hairline effect
- StaleBanner uses flex + span wrapping text: separates layout container from text styling, and leaves room for future icon addition without restructuring
