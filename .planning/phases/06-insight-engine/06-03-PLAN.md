    **rare-events.ts** — export `generateRareEventInsights(): Promise<InsightRow[]>`

    Implement 2 rare event subtypes using PostgreSQL `PERCENT_RANK()` window function. Threshold: top 5% (pct_rank >= 0.95).

    IMPORTANT pitfall to avoid: wrap DISTINCT + ORDER BY in a subquery per STATE.md pattern (PostgreSQL 42P10 error).

    1. **rare_points**: Clippers player scoring performance in top 5% across all player-games in 3-season DB.
       ```sql
       SELECT sub.player_id, sub.game_id, sub.points, sub.pct_rank
       FROM (
         SELECT pb.player_id::text AS player_id, pb.game_id::text AS game_id,
                pb.points,
                PERCENT_RANK() OVER (ORDER BY pb.points ASC) AS pct_rank
         FROM game_player_box_scores pb
         JOIN games g ON g.game_id = pb.game_id
         WHERE g.season_id IN (2022, 2023, 2024)
       ) sub
       JOIN players p ON p.player_id = sub.player_id::bigint
       WHERE sub.pct_rank >= 0.95
         AND p.team_id = $1::bigint   -- LAC team_id
       ORDER BY sub.pct_rank DESC, sub.game_id DESC
       LIMIT 10;
       ```
       - proof_sql: the above query parameterized with $1 = lac_team_id
       - proof_result: the rows returned
       - headline: e.g., "Kawhi Leonard's 45-point game ranks in the top 5% of all NBA performances this 3-season stretch"
       - metric_key: `rare_points_${playerId}_${gameId}`
       - importance: `computeImportance('rare_event', Math.round(row.pct_rank * 100), gameDateMs)`

    2. **rare_assists**: Same structure but for assists (pb.assists).
       - metric_key: `rare_assists_${playerId}_${gameId}`

    For "upcoming game context": query the next game where status != 'final' and (home_team_id = LAC OR away_team_id = LAC). If no upcoming game, use the most recent completed game.

    **league-comparisons.ts** — export `generateLeagueComparisonInsights(): Promise<InsightRow[]>`

    Implement 2 league comparison subtypes using rolling_team_stats.

    1. **clippers_off_rank**: Clippers offensive rating rank in league over last 10 games. Query:
       ```sql
       SELECT COUNT(*) + 1 AS rank, total.total_teams
       FROM rolling_team_stats rts
       CROSS JOIN (SELECT COUNT(DISTINCT team_id) AS total_teams FROM rolling_team_stats WHERE window_size = 10) total
       WHERE rts.window_size = 10
         AND rts.avg_off_rating > (SELECT avg_off_rating FROM rolling_team_stats WHERE team_id = $1::bigint AND window_size = 10)
       ```
       - headline: "Clippers rank 5th in offensive rating over the last 10 games"
       - metric_key: `clippers_off_rank_last10_${asOfDate}`
